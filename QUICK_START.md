# Быстрый старт

## Проблема

Ваш встроенный сервис `quetlink-five.vercel.app` показывает ошибку:
```
Unable to retrieve launch parameters from any known source
```

Это происходит потому, что при запуске в iframe сервис не получает параметры Telegram напрямую.

## Решение (3 шага)

### Шаг 1: Добавьте файл-мост в ваш встроенный сервис

Скопируйте файл `iframe-telegram-bridge.js` из этого репозитория в ваш проект `quetlink-five.vercel.app`.

### Шаг 2: Подключите файл в HTML

Добавьте в начало вашего `index.html` (перед другими скриптами):

```html
<script src="/iframe-telegram-bridge.js"></script>
```

Или если используете сборщик:

```javascript
import './iframe-telegram-bridge.js';
```

### Шаг 3: Готово!

Теперь ваш сервис будет:
- ✅ Получать параметры Telegram от родительского окна
- ✅ Работать в iframe без ошибок
- ✅ Поддерживать все функции Telegram Web App API

## Проверка

1. Откройте родительское приложение в Telegram
2. Перейдите в раздел "ИИ помощник в подарках"
3. Проверьте консоль браузера - не должно быть ошибок
4. Ваш сервис должен работать как обычно

## Отладка

Если что-то не работает:

1. Откройте консоль браузера (F12)
2. Проверьте сообщения с префиксом `[Telegram Bridge]` и `[Parent]`
3. Убедитесь, что файл `iframe-telegram-bridge.js` загружается
4. Проверьте, что сообщения postMessage доходят до iframe
